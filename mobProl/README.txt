-- Informacion Importante --
Los archivos estan codificados en Python.

Se necesita tener python en donde se desea probar
el parser.

Es importante no mover ningun directorio dentro del
directorio principal

El directorio principal contiene:
- lexer.py: Archivo donde se generan los tokens
- parser.py: Archivo que revisa la sintaxis de una entrada
- ply: Directorio que contiene lex y yacc

Se utilizo PLY (Pyhon Lex & Yacc) para la facilidad de uso
con python y para poderlo utiizar en Windows y en MacOS.
